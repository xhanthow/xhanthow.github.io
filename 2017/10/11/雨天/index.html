雨天

发表于 2017-10-11 | 分类于 随笔 | | 本文被瞅了 次

风很大，昨晚就开始刮风了。天气渐凉，方觉一年将至。

南京还是和往常一样，没有秋天。

雨，淅淅沥沥地下着，也仿佛浇灭了自己跑步的欲望。很喜欢刮风的阴天，有一种说不出来的安适感，但是讨厌的雨却夹杂在中间，令人厌烦。夏天，应该过去了吧。自己也想不到，在这个夏天自己会坚持跑步，坚持跑了200多公里，也甩掉了20斤肉，当回去的时候，听见别人说自己瘦了，着实很开心。

情绪是个很奇怪的东西，人也是个很奇怪的动物。

很怕想得太多，又怕想得太少。

成长意味什么了？

怎样又叫做成熟了？

感性与理性谁充当更加重要的角色了？

还有2个多月，这一年就将过去。对，都到了2018年了。时间真的是怪物！很多事都像发生在昨天，记忆尤深，不敢去想。这一年自己好像改变了很多。

看似狂欢，实则寂寞。

莫名期待，终觉伤感。

愿有一方栖息的净土。
